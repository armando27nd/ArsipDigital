Arsip Digital

Sistem manajemen dokumen elektronik ini membantu Anda menyimpan, mengelola, dan mencari arsip secara digital dengan cepat. Aplikasi ini mengubah proses pengarsipan manual menjadi sistem yang terorganisir dan efisien.

Fitur Utama

Unggah dokumen dalam berbagai format file.

Klasifikasi arsip berdasarkan kategori atau departemen.

Pencarian dokumen secara instan menggunakan kata kunci.

Dashboard statistik untuk memantau jumlah dokumen.

Manajemen hak akses pengguna untuk keamanan data.

Teknologi Yang Digunakan

PHP dengan Framework CodeIgniter atau Laravel.

MySQL untuk penyimpanan database.

Bootstrap untuk tampilan antarmuka yang responsif.

Cara Instalasi

Unduh atau klon repositori ini ke komputer Anda.

Pindahkan folder proyek ke direktori server lokal seperti htdocs.

Buat database baru di phpMyAdmin dengan nama arsip_digital.

Impor file database .sql yang tersedia di dalam folder proyek.

Sesuaikan konfigurasi database pada file config atau .env.

Buka browser dan akses alamat localhost/ArsipDigital.

Contoh Penggunaan

Anda dapat menggunakan sistem ini untuk:

Penyimpanan surat masuk dan surat keluar di kantor.

Digitalisasi dokumen aset atau sertifikat penting.

Pusat data referensi untuk organisasi atau lembaga pendidikan.
